4. Tomcat集群会带来什么问题？

+ Session登录信息存储及读取问题  
+ 服务器定时任务并发问题  
+ ....  

###### 解决方案

+ 采用nginx ip hash policy  
 优点：可以不改变现有技术架构，直接实现横向扩展（省事）   
 缺点：  
 1.导致服务器请求（负载）不平均（完全依赖ip hash的结果）  
 2.在ip变化的环境下无法服务
